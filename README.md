# Personal-CA
Tax Calculating Website

A polished, responsive, privacy-first front-end prototype for free individual tax calculations across Bangladesh, India and USA.

## Run

Open `index.html` directly in a browser, or serve the folder locally:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Included features

- Register/login demo with browser-local profile storage
- Dark/light mode
- Responsive futuristic UI for desktop, tablet and mobile
- Adaptive country-specific tax questionnaire
- Bangladesh, India and USA tax calculation models
- Explainable calculation lines and effective rate
- Law/source hyperlinks beside questions and in the tax library
- Save yearly tax calculations to profile and delete them anytime
- Print/download PDF tax report through the browser print-to-PDF flow
- Savings-aware investment and charity suggestions with authentic links
- Tax-reduction suggestions by country
